\# User Journey Map



| Stage | User Action | Touchpoint | Pain Point | Opportunity |

|------|------------|------------|------------|-------------|

| Awareness | Learn about Bitcoin L2 | Website | Too technical language | Simple explanations |

| Onboarding | Install wallet | Wallet App | Complex setup steps | Guided onboarding |

| Setup | Bridge BTC | Bridge Interface | Slow confirmations | Progress indicators |

| Usage | Send transaction | Wallet | Unclear error messages | Better error handling |

| Support | Seek help | Support / Community | Slow responses | Knowledge base |

| Retention | Regular usage | Wallet / Dashboard | Trust issues | Transparency tools |
