# Bitcoin Layer 2 User Journey Map

## Objective
To visualize the end-to-end user experience of a Bitcoin Layer 2 infrastructure
and identify key touchpoints, pain points, and improvement opportunities.

## User Personas
- Developer
- End User
- Merchant

## Journey Stages
1. Awareness
2. Onboarding
3. Setup
4. Usage
5. Support
6. Retention

## Outcome
This project highlights usability gaps and proposes improvements
to increase adoption and user satisfaction.
